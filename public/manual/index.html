HoldMyMap

Manual de Usuario
App de Puntos de Agua para Bomberos Voluntarios

1. Introduccion

HoldMyMap es una aplicacion web para bomberos voluntarios que permite ubicar rapidamente puntos de agua y otros recursos en zonas rurales.

Caracteristicas principales:

  • Busqueda rapida de puntos por nombre
  • Navegacion directa con Google Maps
  • Funciona sin internet (modo offline)
  • Se puede instalar en el celular como app

2. Acceder a la Aplicacion

Desde el navegador

  1. Abrir el navegador (Chrome, Safari, Firefox)
  2. Ingresar la direccion: holdmymap.vercel.app
  3. Ingresar el codigo de grupo proporcionado

Pantalla de inicio con campo de codigo de grupo

3. Instalar en el Celular (Acceso Directo)

La aplicacion se puede instalar como si fuera una app nativa, sin necesidad de tiendas de aplicaciones.

En Android (Chrome)

  1. Abrir holdmymap.vercel.app en Chrome
  2. Tocar el menu ⋮ (tres puntos arriba a la derecha)
  3. Seleccionar "Agregar a pantalla de inicio" o "Instalar aplicacion"
  4. Confirmar tocando "Agregar"
  5. El icono aparecera en tu pantalla de inicio

Paso 1: Menu de Chrome con opcion de instalacion

Paso 2: Confirmar instalacion

En iPhone (Safari)

  1. Abrir holdmymap.vercel.app en Safari
  2. Tocar el boton Compartir (cuadrado con flecha hacia arriba)
  3. Desplazar hacia abajo y tocar "Agregar a inicio"
  4. Tocar "Agregar" en la esquina superior derecha
  5. El icono aparecera en tu pantalla de inicio
Tip: Una vez instalada, la app se abre en pantalla completa sin barras del navegador, como una app normal.

4. Usar la Aplicacion

4.1 Buscar un Punto

  1. En la pantalla principal, veras la lista de todos los puntos
  2. Usa el campo de busqueda para filtrar por nombre
  3. Los resultados se actualizan mientras escribes

Lista de puntos con campo de busqueda activo

4.2 Navegar a un Punto

  1. Encuentra el punto deseado en la lista
  2. Toca el boton Ir a Maps
  3. Se abrira Google Maps con la ruta hacia el punto

Tarjeta de punto mostrando boton "Ir a Maps"

4.3 Agregar un Punto Nuevo

  1. Tocar el boton + Agregar lugar en la parte inferior
  2. Completar el formulario:
    • Nombre: Nombre del punto (obligatorio)
    • Descripcion: Detalles como capacidad de agua, tipo de acceso, etc.
    • Ubicacion: Puedes elegir una de estas opciones:
      • Pegar un link de Google Maps
      • Tocar "Usar mi ubicacion actual"
      • Ingresar coordenadas manualmente
  3. Tocar "Guardar"

Formulario de agregar punto

4.4 Editar o Eliminar un Punto

  1. En la tarjeta del punto, tocar Editar
  2. Modificar los datos necesarios
  3. Tocar "Guardar cambios" o "Eliminar punto"

5. Importar Puntos desde Excel/CSV

Si tienes una lista de puntos en Excel, puedes importarlos masivamente.

Preparar el archivo

El archivo debe tener formato CSV con estas columnas:

Columna Descripcion Obligatorio
Nombre Nombre del punto Si
Detalles Descripcion del punto No
Ubicacion Link de Google Maps o coordenadas Si

Ejemplo de contenido:

Nombre,Detalles,Ubicacion
Campo Los Alamos,20000 lts con bomba,https://maps.google.com/maps?q=-33.49,-64.36
Tanque Municipal,15000 lts con molino,https://maps.google.com/maps?q=-33.51,-64.38

Pasos para importar

  1. Tocar el boton CSV en la parte inferior
  2. Arrastrar el archivo o tocar "Seleccionar archivo"
  3. Verificar la vista previa de los puntos
  4. Tocar Importar X puntos

Pantalla de importacion CSV

6. Modo Sin Internet (Offline)

La aplicacion guarda los puntos en tu dispositivo para funcionar sin conexion a internet.

  • Cuando no hay internet, veras un aviso amarillo en la parte inferior
  • Puedes buscar y ver todos los puntos guardados
  • Los cambios que hagas se sincronizaran automaticamente cuando vuelvas a tener conexion
Importante: Para que el modo offline funcione, debes haber abierto la app al menos una vez con conexion a internet.

7. Compartir el Grupo

Para que otros companeros accedan al mismo grupo de puntos:

  1. Compartir el link directo por WhatsApp:
    holdmymap.vercel.app/CODIGO-GRUPO
  2. O indicarles que ingresen el codigo en la pantalla inicial
Tip: El link directo permite acceder inmediatamente al grupo sin tener que escribir el codigo.

8. Solucion de Problemas

Problema Solucion
No carga la lista de puntos Verificar conexion a internet y recargar la pagina
El boton "Ir a Maps" no funciona Asegurarse de tener Google Maps instalado en el dispositivo
No aparece la opcion de instalar la app Usar Chrome en Android o Safari en iPhone
Los puntos no se sincronizan Esperar a tener buena conexion WiFi o datos moviles
No puedo importar el CSV Verificar que el archivo tenga las columnas correctas (Nombre, Detalles, Ubicacion)
