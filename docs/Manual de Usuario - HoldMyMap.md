# HoldMyMap
## Manual de Usuario

### App de Puntos de Agua para Bomberos Voluntarios

---

## 1. Introducción

HoldMyMap es una aplicación web diseñada para bomberos voluntarios que permite ubicar rápidamente puntos de agua y otros recursos en zonas rurales.

**Características principales:**
- Búsqueda rápida de puntos por nombre
- Navegación directa con Google Maps
- Funciona sin internet (modo offline)
- Se puede instalar en el celular como app

---

## 3. Instalar en el Celular (Acceso Directo)

La aplicación se puede instalar como si fuera una app nativa, sin necesidad de tiendas de aplicaciones.

### En Android (Chrome)

1. Abrir **holdmymap.vercel.app** en Chrome
2. Tocar el menú (tres puntos arriba a la derecha)
3. Seleccionar **"Agregar a pantalla de inicio"** o **"Instalar aplicación"**
4. Confirmar tocando **"Agregar"**
5. El ícono aparecerá en tu pantalla de inicio

[IMAGEN: Menú de Chrome con opción "Agregar a pantalla de inicio"]

[IMAGEN: Ícono de HoldMyMap en pantalla de inicio Android]

### En iPhone (Safari)

1. Abrir **holdmymap.vercel.app** en Safari
2. Tocar el botón **Compartir** (cuadrado con flecha hacia arriba)
3. Desplazar hacia abajo y tocar **"Agregar a inicio"**
4. Tocar **"Agregar"** en la esquina superior derecha
5. El ícono aparecerá en tu pantalla de inicio

## 4. Usar la Aplicación

### 4.1 Buscar un Punto

1. En la pantalla principal, verás la lista de todos los puntos
2. Usa el campo de búsqueda para filtrar por nombre
3. Los resultados se actualizan mientras escribes

[IMAGEN: Lista de puntos con campo de búsqueda]

### 4.2 Navegar a un Punto

1. Encuentra el punto deseado en la lista
2. Toca el botón azul **"Ir con Maps"**
3. Se abrirá Google Maps con la ruta hacia el punto

[IMAGEN: Tarjeta de punto con botón "Ir con Maps"]

### 4.3 Agregar un Punto Nuevo

1. Tocar el botón **"+ Agregar punto"** en la parte inferior
2. Completar el formulario:
   - **Nombre**: Nombre del punto (obligatorio)
   - **Descripción**: Detalles como capacidad de agua, tipo de acceso, etc.
   - **Ubicación**: Puedes:
     - Pegar un link de Google Maps
     - Usar tu ubicación actual tocando "Usar mi ubicación"
     - Ingresar coordenadas manualmente
3. Tocar **"Guardar"**

[IMAGEN: Formulario de agregar punto]

### 4.4 Editar o Eliminar un Punto

1. En la tarjeta del punto, tocar **"Editar"**
2. Modificar los datos necesarios
3. Tocar **"Guardar cambios"** o **"Eliminar punto"**

---

## 5. Importar Puntos desde Excel/CSV

Si tienes una lista de puntos en Excel, puedes importarlos masivamente.

### Preparar el archivo

El archivo debe tener formato CSV con estas columnas:
- **Nombre**: Nombre del punto
- **Detalles**: Descripción (opcional)
- **Ubicacion**: Link de Google Maps o coordenadas

Ejemplo:
```
Nombre,Detalles,Ubicacion
Campo Los Alamos,20000 lts con bomba,https://maps.google.com/maps?q=-33.49,-64.36
```

### Importar

1. Tocar el botón **"CSV"** en la parte inferior
2. Arrastrar el archivo o tocar **"Seleccionar archivo"**
3. Verificar la vista previa
4. Tocar **"Importar X puntos"**

[IMAGEN: Pantalla de importación CSV con vista previa]

## 6. Modo Sin Internet (Offline)

La aplicación guarda los puntos en tu dispositivo para funcionar sin conexión.

- Cuando no hay internet, verás un aviso amarillo en la parte inferior
- Puedes buscar y ver todos los puntos guardados
- Los cambios que hagas se sincronizarán cuando vuelvas a tener conexión

[IMAGEN: Indicador de modo offline]

## 7. Compartir el Grupo

Para que otros compañeros accedan al mismo grupo de puntos:

1. Compartir el link directo: **holdmymap.vercel.app/CODIGO-GRUPO**
2. O indicarles que ingresen el código en la pantalla inicial

Este link se puede compartir fácilmente por WhatsApp.

## 8. Solución de Problemas

| Problema | Solución |
|----------|----------|
| No carga la lista de puntos | Verificar conexión a internet y recargar la página |
| El botón "Ir con Maps" no funciona | Asegurarse de tener Google Maps instalado |
| No aparece la opción de instalar | Usar Chrome en Android o Safari en iPhone |
| Los puntos no se sincronizan | Esperar a tener buena conexión WiFi o datos |
